10. 一次投诉行为会回调我们几次,状态变更是否会推送,其他平台的投诉会推给我们吗

投诉行为会按照结果来回调,直至用戶投诉完成。

状态变更都会回调,包含:货架状态、价格、段位、标题等信息。

其他平台投诉不会返回你们,存在鉴权。

11. 列表接口返回是否都是可租的,
变更接口中返回的数据会存在哪些:

a. 新增的账号数据

b. 状态变更的数据,由租赁中变成可租

3,价格变更或者商品其他信息变更的数据

列表目前返回都是可租的,回调接口中包含的数据参考问题3.

平台新增的货架不进入号池时,不会推送给渠道。

12. 商品删除状态指的是什么

商品删除是指号主将货架从平台中删除,此货架id后续可以不再关心,渠道侧基本收不到。

13. 上报时如果我们这边报错了,会重试吗,需要约定响应格式

目前同步的是,回调接口会存在重试机制,2次。

不需要约定响应格式,根据状态码来判断是否重试,接口超时500这种会重试,返回200不重试。

14. 测试验证阶段,走客诉流程增加标记,不让线上真正的客服处理

测试验证阶段,会整理部分进入人工通道的投诉类型,然后测对应的流程。
