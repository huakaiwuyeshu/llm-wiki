## Page 1

三方渠道整理

简介

在游戏账号租赁市场中,我们与不同类型的三方渠道建立了合作关系,提供定制化的解决方案。我们
的合作伙伴大致可分为三类:

第一类是拥有自身品牌和租号平台的中大型公司。他们具备线下渠道和庞大用戶群体,但在账号资
源、上号技术和⻛控能力方面仍有需求(例如顺网)。

第二类是同行业内的中小型公司。他们拥有一定的开发实力和渠道资源,但在关键环节中,例如账号
资源和防御能力方面存在短板(例如GG租号)。

目前还存在第三类正在探索的合作类型,例如Steambig,在游戏赛道内的主机游戏服务方面存在一定
用戶留存规模,在此基础上展开租号业务合作。

针对不同类型的合作伙伴,我们平台提供灵活的合作模式。我们可为他们提供:

1.
账号资源池:拥有海量优质游戏账号资源,满足租赁需求。

2.
上号技术能力:提供自动化上号系统,确保用戶能够高效便捷地租用账号后一键完成上号进行游戏。

3.
撤单核实能力:针对用戶投诉,调取实际交易数据(账号状态、信息、操作日志等),与投诉理由进行
匹配核查,公正处理撤单请求。

4.
⻛控防护方案:针对账号租赁环节可能出现的⻛险,提供全方位的防御措施。

通过提供上述核心元素,我们的合作伙伴可以充分利用自身优势,同时获得租号玩强大的技术和资源
支持,提升账号租赁业务的盈利能力和用戶体验。同时,我们将持续完善产品能力,为合作伙伴提供
更优质、更全面的增值服务。

## Page 2

业务合作形态

租号玩目前向渠道合作伙伴提供全面的租号解决方案,在提供端游、手游SDK的基础(负责上号)上,提
供两种API对接方案,来传输货架数据

• 本地数据存储:此模式下,合作伙伴服务端向租号玩服务端发起请求,将获取到的货架数据直接存
储到本地系统中,便于后续的货架管理和调用等。

• 在线数据获取:在此模式下,合作伙伴的服务端采用实时请求机制,来获取货架相关的实时信息来
进行展示和后续使用。

商业合作模式

目前在向三方渠道推送货架时,租号玩采取加价运营模式。具体操作为:在租号玩货架上架的租赁价格
基础上,按照既定比例增加一定金额,形成新的租赁价格,推送至各三方平台。每当三方租客在三方
平台中下单租号玩的货架后,我们即可通过加价部分的差价获得收益。

三方下单流程

目前对接三方平台时我们都是让三方平台在我们平台注册一个账号,作为一个租客身份,当三方租客
在对应三方平台下单后,对应三方平台判断是我们平台的货架后,通过调取我们平台的下单接口,通
过中转账号进行下单。
