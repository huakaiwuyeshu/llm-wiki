1 概述
PC 端游上号 SDK 是针对租号平台所有游戏开发的一个上号功能的 SDK。为了方便调试,目前暂时
没有集成防御模块。部分上号功能可能会受到影响,所以此模块仅限于调试接口使用。

本文档所描述的内容适用于任何需要使用租号玩 PC 端游戏账号进行上号的第三方公司。如果您的

(1)必须从主进程加载 zhplatform.dll
(2)主程序有 Windows 签名
(3)主进程有管理员权限

## Page 6

货架共享第三方 API 接口文档
5. 开始游戏接口(ClickStartBtn)
从 zhplatform.dll 获取 ClickStartBtn 接口。只有等初始化完成之后才会开始检测环境、启动游戏客户
端、上号。这个过程一般和主程序交互比较频繁,主要是检测进度和订单时候被投诉等情况。
6. CSGO 游戏开始游戏接口(ZstartCsgo)
从 zhplatform.dll 获取 ZstartCsgo 接口。开始上号前,如果货架支持双端(steam 和蒸汽平台)登录,
那么就弹出一个选择框,让用户先择 steam 或者蒸汽。用户选择之后,将对应的客户端 ID 传给接
口。
7. CSGO 游戏平台 开始登录接口(ZstartCsgoPlatform)
从 zhplatform.dll 获取 ZstartCsgoPlatform 接口。用户下单之后,如果用户的订单支持 5E 或者完美
平台登录。就提供一个点击按钮。让用户选择是使用完美平台还是 5E 平台上号
